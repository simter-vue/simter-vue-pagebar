# simter-vue-pagebar

A [Vue] pagebar component.

## Develop

```shell
$ yarn install
$ yarn start
```

> `'yarn start'` use [Parcel] to run `'demo/index.html'`.

## Build

```shell
$ yarn build
```

> Build to `'dist'` directory by [Rollup].


[Vue]: https://vuejs.org
[Parcel]: https://parceljs.org
[Rollup]: https://rollupjs.org
